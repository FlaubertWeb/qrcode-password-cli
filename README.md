  <h1>💻 Projeto Node.js: Utilidades para E-commerce</h1>

  <p>Esse projeto foi desenvolvido como parte do meu aprendizado em Node.js. A proposta foi criar um kit de utilidades para e-commerce com foco em escalabilidade e organização por camadas.</p>

  <h2>📚 O que eu já precisava saber antes</h2>
  <p>Pra conseguir desenvolver esse projeto, foi importante ter uma base em:</p>
  <ul>
    <li>Lógica de programação</li>
    <li>JavaScript básico</li>
    <li>Node.js básico</li>
    <li>node_modules e como funcionam os imports</li>
    <li>Instalação de pacotes via NPM</li>
    <li>Uso de variáveis de ambiente com <code>.env</code></li>
  </ul>

  <h2>🛠️ O que eu aprendi</h2>
  <ul>
    <li>Como gerar QR Codes com Node.js</li>
    <li>Como lidar com várias dependências num mesmo projeto</li>
    <li>Como organizar o código em camadas (separando prompts, serviços, etc.)</li>
  </ul>

  <h2>🎯 Resultado final</h2>
  <p>Com esse projeto finalizado, consegui entender como:</p>
  <ul>
    <li>Criar aplicações Node.js com múltiplas funcionalidades</li>
    <li>Adicionar novas features sem bagunçar o código</li>
    <li>Fazer o terminal interagir com o usuário via prompt</li>
  </ul>

  <h2>🚀 Como rodar</h2>
  <pre><code>npm install
node src/index.js</code></pre>
